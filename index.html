Hola, soy Obed

Soy un apasionado desarrollador con experiencia en el diseño y creación de aplicaciones web y móviles. Además, cuento con habilidades en soporte técnico y hardware, lo que me permite abordar problemas tecnológicos de manera integral. Actualmente, estoy en la recta final de mi formación como Ingeniero en Gestión de Software, lo que complementa mis conocimientos técnicos con una visión estratégica para la resolución de problemas y el desarrollo de proyectos innovadores.

Proyectos

Proyecto 1


Taskify App

Creé Taskify, una app para organizar tareas. Tiene una interfaz fácil de usar donde puedes añadir, editar y eliminar tareas. Puedes asignarles un título, descripción y establecer fechas. También incluye un calendario para visualizar las tareas de forma clara. Con esta app, puedes gestionar tu día a día de manera sencilla y eficiente.


Proyecto 2


Sopi letras

Sopi Letras es un juego de sopa de letras que desarrollé como parte de un proyecto en mi universidad. El jugador busca palabras en una cuadrícula tras seleccionar su nombre, color y dificultad. Durante la partida, un temporizador mide el tiempo y las palabras encontradas se destacan. Al finalizar, se muestra un resumen con el progreso logrado.



Proyecto 3


Polainas Candys

Polaina's Candys es una aplicación desarrollada como proyecto universitario para gestionar una tienda de dulces. Los usuarios pueden iniciar sesión, buscar dulces por categorías y consultar precios. Los administradores tienen acceso a gestionar sucursales y productos. La app es intuitiva y está diseñada para facilitar la experiencia del cliente y la administración.



Proyecto 4


Logs

Esta aplicación la realicé en la universidad. Es una herramienta de gestión de usuarios y registros, donde puedo agregar, ver y editar la información de los usuarios. Además, tiene una sección para registrar eventos, mostrando los detalles de cada uno, como el tipo de evento y la dirección IP asociada. Todo esto se presenta en una interfaz sencilla y limpia.



Proyecto 5


Pasteleria

La página que desarrollé sobre pasteles fue parte de un proyecto de ingeniería que realicé en la universidad. Este proyecto consistía en un CRUD (crear, leer, actualizar, eliminar) para gestionar pasteles en un sistema. Además, estaba conectada a un host web para poder ser accesible desde cualquier lugar.
